# BB8
Model BB8 robot as seen in Star Wars. 

Note: This project is still in progress.

## Parts

* 13" Hamster Ball

* Raspberry Pi 3B+

* Arduino Uno

* 4 wheels

* 2 motors

* L298N Motor driver board or similar
<img src = "images/L298N_board.jpg" width = 200>

* Battery pack
